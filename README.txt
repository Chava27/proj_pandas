En este repositorio hay dos porgramas los cuales leen una base de datos de Excel(fiestas_data.xlsx). El programa proyecto_pandas.py se encarga de utilizar la liberia de pandas y numpy para graficar y obtener datos estadísticos de la base de datos.

Una vez que se pudo visualizar la información de una mejor manera fue momento de crear un programa donde aplicamos el algoritmo K-means. Para este programa decidimos implementar k-means en distintas comparacíones de la base de datos.

El primero de ellos fue comparar el número total de invitados con los meses del año, en el que primero generamos una lista de los meses convertidos a números para poder compararlos. Para este caso se decidio obtener 4 clusters. Se descubrió que las separación de variables se realiza como una tabla lineal.

El segundo de estos fue comparar el número total de invitados con la cantidad total gastada en la comida. Para este caso decidimos obtener 3 clusters para encontrar del promedio de gastos para cada sección. Un dato que descubrimos fue que la mayoría de los datos se encontraban agrupados en la segunda sección.

El tercero de estos fue comparar el número total de invitados con la cantidad total gastada en la fiesta. Para este caso decidimos obtener 4 clusters. Cabe aclarar de que en el caso 1 y 2 se gráfico sin los clusters para poder observar la diferencia, sin embargo para este último no se realizo ya que no era muy necesario.
