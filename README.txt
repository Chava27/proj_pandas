En este repositorio hay dos porgramas los cuales leen una base de datos de Excel(fiestas_data.xlsx). El programa proyecto_pandas.py se encarga de utilizar la liberia de pandas y numpy para graficar y obtener datos estadísticos de la base de datos.

Una vez que se pudo visualizar la información de una mejor manera fue momento de crear un programa donde aplicamos el algoritmo K-means. Para este programa decidimos implementar k-means en distintas comparacíones de la base de datos. El primero de estos fue comparar el numero total de invitados con la cantidad total gastada en la comida. Para este caso decidimos obtener 3 clusters para encontrar eel promedio de gastos para cada sección. Un dato que descubirmos fue que la mayoria de los datos se encontraban agrupados en la segunda sección.
